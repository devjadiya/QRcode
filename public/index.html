Barcode Scanning API

This is a RESTful API built with Express.js for scanning barcodes from Base64 encoded images using Quagga2 for detection and Sharp for preprocessing.

View on GitHub

Barcode Detection

Supports multiple barcode formats (EAN, UPC, Code 128, Code 39, Code 93, and I2of5) with high accuracy.

Base64 Image Input

Accepts Base64 encoded image data for barcode scanning with easy integration into your projects.

Image Preprocessing

Enhances image quality by resizing, converting to grayscale, and improving contrast for better barcode detection.

Error Handling

Provides helpful error messages for invalid or unsupported inputs to guide you through the scanning process.
